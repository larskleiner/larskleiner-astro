# Contemporary Redesign Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Redesign larskleiner.de in a Studio direction — white, system-ui typography, 2-column card grid for social links, Permanent Marker reserved for card titles and inner page headings, consistent header and inner page styling.

**Architecture:** CSS is fully rewritten as semantic scoped rules (no utilities). A new `Header.astro` component replaces `Nav.astro`, combining the profile photo, name, tagline and language switcher in one bar. All pages use this shared header. The old Bootstrap utility class pattern is removed entirely; CSS Grid handles the card layout.

**Tech Stack:** Astro 6, TypeScript, plain CSS, `astro:assets` Image
