# Speed, SEO & Accessibility Optimisation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove Bootstrap, self-host the Google Font, add SEO meta tags, and fix accessibility gaps across the Astro personal website.

**Architecture:** All changes are in existing source files plus one new font file. No new components or pages. Tasks are ordered so each commit leaves the site in a buildable state: font self-hosting first (keeps the existing Bootstrap CSS intact), then Bootstrap replacement, then SEO, then accessibility.

**Tech Stack:** Astro 6, TypeScript, plain CSS

## Task 1: Self-host Permanent Marker font

**Files:**
- Create: `public/fonts/PermanentMarker-Regular.woff2`
- Modify: `src/styles/main.css`
- Modify: `src/layouts/BaseLayout.astro`

- [ ] **Step 1: Download the font file**

Run from `~/Sites/larskleiner/larskleiner-astro/`:

```bash
mkdir -p public/fonts

FONT_URL=$(curl -s \
  -H "User-Agent: Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 Chrome/120.0.0.0 Safari/537.36" \
  "https://fonts.googleapis.com/css2?family=Permanent+Marker&display=swap" \
  | grep -o 'https://fonts.gstatic.com[^)]*\.woff2' | head -1)

echo "Downloading: $FONT_URL"
curl -L -o public/fonts/PermanentMarker-Regular.woff2 "$FONT_URL"
```

Expected: `public/fonts/PermanentMarker-Regular.woff2` exists and is non-zero bytes:

```bash
ls -lh public/fonts/PermanentMarker-Regular.woff2
```

Expected output: file size around 20–40 KB.

- [ ] **Step 2: Add `@font-face` to main.css**

Add the following block at the **top** of `src/styles/main.css` (before the existing `body` rule):

```css
@font-face {
  font-family: 'Permanent Marker';
  src: url('/fonts/PermanentMarker-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
```

- [ ] **Step 3: Remove Google Fonts links from BaseLayout**

In `src/layouts/BaseLayout.astro`, remove these three lines from `<head>`:

```html

```

- [ ] **Step 4: Verify build succeeds**


Expected: `0 errors` from check, build completes successfully. The Permanent Marker font should still render correctly (served from `/fonts/` now).

- [ ] **Step 2: Update BaseLayout.astro frontmatter**

Replace the entire frontmatter section of `src/layouts/BaseLayout.astro` with:

```astro
---
import { getAbsoluteLocaleUrl } from 'astro:i18n'
import '../styles/main.css'
import { useTranslations } from '../i18n/translations'

interface Props {
  title: string
  lang: string
  pagePath: string
  description?: string
}

const { title, lang, pagePath, description: descriptionProp } = Astro.props
const t = useTranslations(lang)
const description = descriptionProp ?? t.description
const canonicalUrl = getAbsoluteLocaleUrl(lang, pagePath)
const deUrl = getAbsoluteLocaleUrl('de', pagePath)
const enUrl = getAbsoluteLocaleUrl('en', pagePath)
---
```

- [ ] **Step 3: Update BaseLayout.astro `<head>`**

```astro
    <title>{title}</title>
    <meta name="description" content={description} />
    <link rel="canonical" href={canonicalUrl} />
    <link rel="alternate" hreflang="de" href={deUrl} />
    <link rel="alternate" hreflang="en" href={enUrl} />
    <link rel="alternate" hreflang="x-default" href="https://www.larskleiner.de/de/" />
    <meta property="og:type" content="website" />

    <meta property="og:url" content={canonicalUrl} />
```
